# API Endpoints

## HTML API

### Root

* GET / - loads React web app

## JSON API

### Users

* POST /api/users

### Session

* POST /api/session
* DELETE /api/session
* GET /api/session

### Gigs

* GET /api/gigs
* POST /api/gigs
* GET /api/gigs/:id
* DELETE /api/gigs/:id
* PATCH /api/gigs/:id

### Orders

* GET /api/orders
* POST /api/orders
* DELETE /api/order/:id

### Carts

* GET /api/carts
* PATCH /api/carts
